Quellfilter, die ein Clip erzeugen

AVISource / OpenDMLSource / AVIFileSource / WAVSource öffnet eine Video-Datei
DirectShowSource öffnet eine Videodatei mittels DirectShow
SegmentedAVISource / SegmentedDirectShowSource lädt automatisch bis zu 100 AVI-Segmente
BlankClip / Blackness erzeugt ein einfarbiges Clip mit den angegebenen Eigenschaften

Filter zur Bildbearbeitung

AddBorders fügt einen schwarzen Rand um das Bild dazu
Blur / Sharpen Einfache 3x3-kernel Schärfe- und Unschärfe-Filter
ConvertToRGB / ConvertToYUY2 / ConvertBackToYUY2 / ConvertToRGB32 / ConvertToRGB24 AviSynth kann intern mit zwei Farbformaten RGB und YUY2 rechnen, diese Filter dienen zur Umwandlung dazwischen
Crop / CropBottom schneidet den Rand eines Bildes ab
GeneralConvolution Allgemeine Konvolution.
Greyscale Umwandlung in ein Grauwert-Video
Layer / Mask / ResetMask / ColorKeyMask Überlagerung zweier Videos, diese müssen nicht gleich gross sein
Letterbox färbt den Rand schwarz ein
Levels ändert die Helligkeitsverteilung und den Gammawert.
MergeChroma / MergeLuma Mischen der Farb- oder Helligkeitswerte zweier gleich grosser Videos
ReduceBy2 / HorizontalReduceBy2 / VerticalReduceBy2 Verkleinerung auf die Hälfte.
BilinearResize / BicubicResize / LanczosResize / PointResize Vergrössern und Verkleinern mit verschiedenen Methoden
RGBAdjust Verändern der einzelnen Farbkomponenten
SpatialSoften / TemporalSoften Rauschfilter durch adaptives Mischen von Pixel
Tweak Verändern von Farbton, Sättigung, Helligkeit und Kontrast

Filter zum Scheiden

AlignedSplice / UnalignedSplice Zusammenfügen von Video-Clips (mit oder ohne Anpassung des Audios)
AssumeFPS / ChangeFPS / ConvertFPS Ändern der Bildwiederholrate
DeleteFrame einen Frame löschen
Dissolve ähnlich AlignedSplice, allerdings mit einem Bereich in dem die Videos gemischt werden
DuplicateFrame verdoppelt einen Frame
FadeOut / FadeOut2 Ausblenden von Video und Audio
FreezeFrame Ersetzen von Frames
Loop einen Teil eines Videos wiederholen
SelectEven / SelectOdd jeden zweiten Frame verwenden (meist bei Videos, die aus Fields bestehen)
SelectEvery die allgemeine Form von SelectEven und Pulldown
Trim Schneiden eines Videos

Filter zum Bearbeiten von Interlaced Video

AssumeFrameBased / AssumeFieldBased zum Umschalten wie das Video behandelt wird
AssumeTTF / AssumeBFF zum Bestimmen der Halbbild-Reihenfolge
Bob bob-deinterlacen
ComplementParity Ändern der Halbbild-Reihenfolge
DoubleWeave ähnlich wie Weave, es wird aber die doppelte Frameanzahl erzeugt
Interleave Verzahnen von Clips
Pulldown zwei von fünf Frames
SeparateFields ein Interlaced-Bild in die Fields mit der doppelten Framerate aufsplitten
SwapFields Tauschen der Fields
Weave das Gegenteil von SeparateFields

Audio Filter

Amplify / AmplifydB Ändern der Lautstärke
AssumeSampleRate Anpassen der Sample-Frequenz und damit Ändern der Abspielgeschwindigkeit
AudioDub einfügen des Audios
ConvertToMono zu Mono wandeln
ConvertAudioTo16bit zu 16 bits wandeln
DelayAudio die Audiospur verzögern
EnsureVBRMP3Sync Asynchrone Wiedergabe von mp3-AVI's vermeiden
GetLeftChannel / GetRightChannel Auswahl eines Kanals
KillAudio Entfernt das Audio
LowPassAudio / HighPassAudio Hochpass und Tiefpass-Filter
MixAudio Mischen vom Audio
MonoToStereo Aus zwei Mono ein Stereo erzeugen
Normalize Auf maximale Lautstärke erhöhen
ResampleAudio echtes Umrechnen der Sample-Frequenz

Spezielle und exotische Filter

Animate ein Meta-Filter, welches andere Filter mit kontinuierlich veränderten Werten aufruft
FixBrokenChromaUpsampling spezielles Filter z.B. für den MS DV Codec
FlipVertical stellt das Video auf den Kopf
Reverse Rückwärts abspielen
Subtitle eine Zeile Text einfügen

Debug- und Test-Filter

ColorBars erzeugt ein Videoclip mit einem Testbild und Testton
Compare Vergleicht zwei Videos
Histogram fügt ein Helligkeits-Histogram hinzu
MessageClip erzeugt ein Clip mit einer Nachricht als Text
ShowFrameNumber / ShowSMPTE blendet die Frame-Nummer oder den Zeitcode ein
StackHorizontal / StackVertical reiht Videos nebeneinander
Subtract die Differenz zweier Videos
Version erzeugt ein Video mit der Versionsnummer und der Copyright-Information
