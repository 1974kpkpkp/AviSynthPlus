Amplify / AmplifydB

Amplify(clip, float amount)
Amplify(clip, float left, float right)
AmplifydB(clip, float amount)
AmplifydB(clip, float left, float right)

Amplify verändert die Lautstärke um den Faktor amount. Beide Kanäle (left und right) können getrennt angegeben werden. AmplifydB ist das gleiche, nur sind die Werte in dB. Negative dB-Werte oder Faktoren zwischen 0 und 1 können zum Reduzieren der Lautstärke verwendet werden. Negative Werte drehen die Phase um 180°.

Audio samples werden automatisch zu 16 bit konvertiert.
