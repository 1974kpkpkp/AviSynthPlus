Blur / Sharpen

Blur(clip, float amount)
Sharpen(clip, float amount)

Diese zwei Filter sind einfache 3x3-Kern Unschärfe- bzw. Schärf-Filter. Der größte erlaubte Wert ist 1.58, das entspricht einer gleichmässigen 3x3-Pixel Unschärfe. Ein Wert von 1.0 entspricht einem (1/4,1/2,1/4) Kern. Für gaussche Unschärfen mit grösserem Radius können mehrere Befehle Blur(1.0) hintereinander gereiht werden.

Negative Argumente von Blur schärfen das Bild, da Sharpen( x ) nur eine andere Schreibweise ist für Blur( -x ). Der kleinste erlaubte Wert für Blur ist -1.0, der größte für Sharpen ist 1.0.
