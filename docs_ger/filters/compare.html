Compare

Compare(clip_filtered, clip_original, string "channels", string "logfile", bool "show_graph")

Compare ermöglicht einen Vergleich zwischen einem gefilterten Clip und dem Original. Dazu wird Text und eine Grafik (wenn show_graph = true) im Bild ausgegeben (Mean Absolute Difference, Mean Difference & PSNR). Der string channels ist eine Kombination aus R,G,B[,A] oder Y,U,V , je nach dem Farbraum des Quell-Clips. Ein leerer String wird als "YUV" oder "RGB" interpretiert. Wenn logfile vorhanden ist, erfolgt die Ausgabe des Textes nur in diese Datei, dies ist viel schneller und besonders nützlich beim Vergleich von längeren Clips.

[funktioniert noch nicht in Version 2.5]

Beispiele:

# Zeigt die Unterschiede am Bildschirm an:
Compare(clip1,clip2)

# Erzeugt eine Logdatei:
Compare(clip1,clip2,"","compare.log")

# Vergleicht nur das Farbsignal von zwei YUY2 Clips: Compare(clip1,clip2,"UV")
