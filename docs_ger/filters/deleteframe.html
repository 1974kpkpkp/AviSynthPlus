DeleteFrame

DeleteFrame(clip, int frame)

DeleteFrame löscht den im Argument angegebenen Frame. Der Ton wird nicht verändert, daher entsteht eine merkbare Asynchronität zwischen Audio und Video, wenn der Filter zu oft verwendet wird.

Wenn man einen Teil von a bis b mitsamt dem Audioteil löschen möchte, geht man so vor:
Trim(0,a-1) ++ Trim(b+1,0)
