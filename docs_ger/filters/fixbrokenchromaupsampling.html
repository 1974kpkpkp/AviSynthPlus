FixBrokenChromaUpsampling

FixBrokenChromaUpsampling(clip)

Der Microsoft DV Codec wandelt die Farbkanäle nicht ganz richtig um, mit diesem Filter kann das korrigiert werden.
Dieses Filter muss gleich nach DirectShowSource verwendet werden (eben beim MS DV Codec oder wenn man farbverschobene Linien bei interlaced-Bildern sieht).
