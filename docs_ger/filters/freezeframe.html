FreezeFrame

FreezeFrame(clip, int first_frame, int last_frame, int source_frame)

Dieser Filter ersetzt alle Frames zwischen first-frame und last-frame mit einer Kopie von source-frame. Der Ton wird nicht verändert. Dies kann zum Ausbessern von Fehlern verwendet werden, bei denen man für defekte Frames ähnliche, fehlerfreie Frames verwenden kann.
