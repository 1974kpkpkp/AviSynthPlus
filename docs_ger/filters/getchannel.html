GetLeftChannel / GetRightChannel

GetLeftChannel(clip)
GetRightChannel(clip)

Hat als Ergebnis den linken oder rechten Kanal eines Stereo-Signals. Das Audio wird ausserdem auf 16 bit umgewandelt. 

Beispiel:
# entfernt den rechten Kanal, erzeugt also ein Mono-Clip mit dem Audio vom linken Kanal:
stereo=WavSource("c:\afx-ab3_t4.wav")
return GetLeftChannel(stereo)
