GetChannel

GetChannel(clip, int ch1 [, int ch2, ...])

Seit der Version 2.5 ersetzt GetChannel GetLeftChannel, indem der linken Kanal wiedergeben wird, und GetRightChannel, indem ein oder mehr Kanäe eines Mehrkanalsignales wiedergegeben wird. Die Kanäle sind folgendermaßen nummeriert:

Stereo:
1 linker Kanal
2 rechter Kanal

6-kanälige WAV Datei (erzeugt aus AC3 5.1):
1 vorderer linker Kanal
2 vorderer rechter Kanal
3 Center Kanal vorne
4 hinterer linker Kanal
5 hinterer rechter Kanal
6 LFE (Subwoofer)

Beispiele:

# Entfernt den rechten Kanal und gibt einen Mono Ton bestehend aus dem linkem Kanal zurück:
video = AviSource("C:\Dateiname.avi")
stereo = WavSource("C:\Dateiname_wav.wav")
mono = GetChannel(stereo, 1)
return AudioDub(video, mono)

# Konvertiert eine AVI mit unkompremiertem Mehrkanal-WAV Ton in ein Stereo Signal:
video = AviSource("C:\Dateiname.avi")
audio = WavSource("C:\Dateiname_wav.wav")
stereo = GetChannel(audio, 1, 2)
return AudioDub(video, stereo)
