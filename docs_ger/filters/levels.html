Levels

Levels(clip, int input_low, float gamma, int input_high, int output_low, int output_high)

Mit diesem Filter wird die Helligkeit, der Kontrast und der Gamma-Wert verändert. Mit den Parametern input_low und input_high wird eingestellt, welche Werte des Eingangs-Videos als völlig schwarz und völlig weiss betrachtet werden; mit output_low und output_high werden die Werte eingestellt, die beim entstehenden Video als Schwarz und Weiss gelten sollen; der Parameter gamma bestimmt den Grad der Nichtlinearität bei der Umwandlung, oder in Formeln:

output = [(input - input_low) / (input_high - input_low)]1/gamma (output_high - output_low) + output_low

Dies ist einer der Filter, für die eine graphische Oberfläche ganz nett wäre. Da es dafür aber keine gibt (zumindestens nicht in AviSynth selber), habe ich den Filter kompatibel mit den von VirtualDub gemacht, zumindesten wenn der Clip in RGB ist. Man kann dafür die Parameter in VirtualDub's Levels Dialog ermitteln, mit diesem Levels Filter verwenden und bekommt die gleichen Ergebnisse.

Beim Bearbeiten im YUY2-Farbraum wird von Levels nur die Helligkeitsinformation durch Gamma korrigiert, da diese Korrektur eigenltich ein RGB-Konzept ist. Bei gamma = 1.0 sollte der Filter aber in RGB und YUY2 den gleichen Effekt haben.

Beispiele:

# keine Veränderung
Levels 0,1,255,0,255

# Gamma-Korrektur für eine Anzeige in einer hellen Umgebung
Levels 0,1.3,255,0,255

# das Bild invertieren (ein Photo-Negativ erzeugen)
Levels 0,1,255,255,0
