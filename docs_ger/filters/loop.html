Loop

Loop(clip, int "times", int "start", int "end")

Loop wiederholt das Segment von start frame bis end frame so oft wie durch times angegeben.

times (Standardwert = -1): so oft wird die Schleife durchlaufen
start (Standardwert = 0): der Frame vom Clip, bei dem die Schleife startet
end (Standardwert = framecount(clip)): der Frame vom Clip, bei dem die Schleife endet

Beispiele:

Loop(10,100,110) # Wiederholt Frame 100 bis 110 vom aktuellern Clip 10 mal
Loop()           # wiederholt den aktuellen Clip praktisch unendlich oft
Loop(10)         # wiederholt den Clip 10 mal
Loop(10,20,29)   # wiederholt Frame 20 bis 29 10 mal, bevor weitergespielt wird
                 # das Clip wird um 90 Frames länger
Loop(0,20,29)    # Frames 20 bis 29 löschen
                 # das Clip wird um 10 Frames kürzer
Loop(-1,20,29)   # Frame 20 bis 29 wird unendlich oft wiederholt
