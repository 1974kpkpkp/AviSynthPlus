MessageClip

MessageClip(string message, int "width", int "height", bool "shrink", int "text_color", int "halo_color", int "bg_color")

Es wird ein Clip mit einer Textzeile generiert. Wird zum Beispiel intern für die Fehlermeldungen verwendet. Eine Arial-Schriftart wird verwendet, die Grösse zwichen 24 points und 9 points wird automatisch gewählt, damit die Schrift in das Video passt. Die Breite und Höhe kann mit width und height angegeben werden. Bei shrink=true wird die Grösse vermindert, damit der Text genau hinein passt.
text_color, halo_color und bg_color geben die Farben der Schrift, des Schrift-Halos und des Hintergrundes an.
