MonoToStereo

MonoToStereo(left_channel_clip, right_channel_clip)

Wandelt zwei Mono-Signale in ein Stereo-Signal um. Das kann zum Beispiel verwendet werden, wenn beide Kanäle getrennt bearbeitet wurden und dann kombiniert werden sollen.

Die beiden Clips müssen die gleiche Sample-Rate haben (ResampleAudio bei Problemen).Wenn eine Quelle schon Stereo ist, wird das Signal vom entsprechenden Kanal genommen (der linke vom left channel clip, und umgekehrt). Audio wird zu 16 bit konvertiert

  
Beispiel:
# Kombinert zwei WAV-Quellen zu einem Stereo-Signal:
left_channel=WavSource("c:\left_channel.wav")
right_channel=WavSource("c:\right_channel.wav")
return MonoToStereo(left_channel, right_channel)
