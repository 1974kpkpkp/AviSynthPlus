MonoToStereo

MonoToStereo(clip "left_channel_clip", clip "right_channel_clip")

Wandelt zwei Mono-Signale in ein Stereo-Signal um. Das kann zum Beispiel verwendet werden, wenn beide Kanäle getrennt bearbeitet wurden und dann kombiniert werden sollen.

Die beiden Clips müssen die gleiche Samplingrate haben (ResampleAudio bei Problemen). Wenn eine Quelle schon Stereo ist, wird das Signal vom entsprechenden Kanal genommen (der linke vom left_channel_clip, und umgekehrt). Der Ton wird als Nebeneffekt zu 16 Bit konvertiert.

Bei AviSynth v2.5 wird anstatt dieses Filters einfach automatisch MergeChannels genommen.

Beispiel:

# Kombinert zwei WAV-Quellen zu einem Stereo-Signal:
left_channel = WavSource("C:\Dateiname1.wav")
right_channel = WavSource("C:\Dateiname2.wav")
return MonoToStereo(left_channel, right_channel)
