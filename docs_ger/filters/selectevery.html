SelectEvery

SelectEvery(clip, int step_size[, int offset1[, int offset2[,...]]])

SelectEvery ist die Verallgemeinerung von Filter wie SelectEven und Pulldown.

Das wird am besten durch Beispiele erklärt: 

SelectEvery(clip,2,0)      # das gleiche wie SelectEven(clip)
SelectEvery(clip,2,1)      # das gleich wie SelectOdd(clip)
SelectEvery(clip,10,3,6,7) # verwendet Frames 3,6,7,13,16,17,... vom Quell-Clip
SelectEvery(clip,9)        # verwendet Frames 0,9,18,... (als zweites Argument wird 0 angenommen)

Oder so: 
# auf ein 24fps progressives Quellclip 3:2 Pulldown anwenden,
# was ein 30fps interlaced Clip ergibt
AssumeFrameBased
SeparateFields
SelectEvery(8, 0,1, 2,3,2, 5,4, 7,6,7)
