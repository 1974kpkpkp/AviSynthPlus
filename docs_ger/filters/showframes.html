ShowFrameNumber

ShowFrameNumber(clip, bool "scroll")

ShowFrameNumber schreibt auf jeden Frame dessen Nummer, so wie AviSynth sie durchnumeriert. Das kann manchmal beim Erstellen der Skripts hilfreich sein. Wenn danach andere Filter angewendet werden, behandeln diese den Text als normale Bildinformation, daher kann der Text unscharf werden.

Wenn scroll=true wird die Nummer nur einmal in das Video gezeichnet und springt von oben nach unten, bei scroll=false wird die Nummer sooft wie möglich auf die rechte Seite des Videos geschrieben.

ShowSMPTE

ShowSMPTE(clip, float fps)

ShowSMPTE ist ähnlich wie ShowFrameNumber, zeigt aber einen SMPTE-Zeitcode an (Stunden:Minuten:Sekunden:Frames). Das Argument fps muss angegeben werden und muss 24, 25, 30 oder 29.97 sein.

Vorsicht: beide Filter sind durch das Zeichnen des Textes langsam
