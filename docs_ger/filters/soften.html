SpatialSoften / TemporalSoften

SpatialSoften(clip, int radius, int luma_threshold, int chroma_threshold)
TemporalSoften(clip, int radius, int luma_threshold, int chroma_threshold)

Die Filter SpatialSoften und TemporalSoften entfernen Rauschen vom Video, indem gezielt Pixels gemischt werden. Diese Filter können wahre Wunder verbringen, allerdings werden feine Details verwischt, wenn die Filterwirkung zu stark eingestellt wird. Weiters sind sie ziemlich langsam, besonders bei einem grossen Wert von radius, sie sollten daher beim Testen noch nicht verwendet werden.

SpatialSoften ersetzt jedes Pixel mit dem Durchschnitt aller Nachbarpixel (innerhalb einem Abstand von radius in x- und y-Richtung), die sich vom Zentral-Pixel um weniger als der Grenzwert unterscheiden. Die Grenzwerte könen für Helligkeit (luma_threshold) und Farbe (chroma_threshold) getrennt angegeben werden.

TemporalSoften ist ähnlich, allerdings werden hier Pixel in Nachbar-Frames (innerhalb eines zeitlichen Abstandes radius) betrachtet. Dieses Filter ist nicht so effektiv wie SpatialSoften.

Am besten man probiert verschiedene Grenzwerte einfach aus. Bei einem Wert von Null erfolgt keine Filterung.

TemporalSoften behandelt Helligkeit und Farbe unabhängig, SpatialSoften wirkt nur, wenn Helligkeit und Farbe innerhalb dem Grenzwert liegen.

Diese Filter funktionieren nur mit YUY2.
