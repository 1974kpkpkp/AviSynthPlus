Subtitle

Subtitle(clip, string text, int "x", int "y", int "first_frame", int "last_frame", string "font", int "size", int "text_color", int "halo_color")
Subtitle(clip, string "text")

Dieses Filter schreibt eine Zeile Text (mit Anti-Aliasing = Kantenglättung) auf einen Bereich von Frames. Wenn man mehr als einen Untertitel braucht oder mehrzeiligen Text braucht, muss man mehrere Subtitle-Filter hintereinander fügen. Dieses Filter ist eher nur für einfache Anwendungen gedacht.

Die Text- und Halo-Farbe müssen als Hexadezimale RGB-Werte (wie in HTML, nur mit $ statt #) angegeben werden

Die kurze Form ist nützlich, wenn nur eine schnelle Beschriftung der Frames stattfinden soll, wobei das genaue Aussehen egal ist.

Dieses Filter wird intern auch für Version, zum Rückmelden von Fehlermeldungen und bei ShowFrameNumber verwendet.
