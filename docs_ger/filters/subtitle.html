Subtitle

Subtitle(clip, string text, int "x", int "y", int "first_frame", int "last_frame", string "font", int "size", int "text_color", int "halo_color" int "align" int "spc")
Subtitle(clip, string "text")

Dieser Filter schreibt eine Zeile Text (mit Anti-Aliasing = Kantenglättung) auf einen Bereich von Frames. Wenn man mehr als einen Untertitel braucht oder mehrzeiligen Text braucht, muss man mehrere Subtitle-Filter hintereinander fügen. Dieser Filter ist eher nur für einfache Anwendungen gedacht.

Die Text- und **Halo-Farbe müssen als Hexadezimale RGB-Werte (wie in HTML, nur statt # - dies ist ja ein Kommentar - ein $ ) angegeben werden.

Die kurze Form ist nützlich, wenn nur eine schnelle Beschriftung der Frames stattfinden soll, wobei das genaue Aussehen egal ist.

Dieser Filter wird intern auch für Version, für Fehlermeldungen und bei ShowFrameNumber verwendet.

align: 4 = links, Grundlinie; 5 = zentriert, Grundlinie (wenn x=-1 ist); die Nummer der Ausrichtung entspricht der Anordnung der Nummer 1-9 **im rechten Zahlenfeld einer Tastatur).

spc: 0 = **no expansion / contraction of text spacing)

**x=8 if align left (1,4,7); x=center if align center(2,5,8); x=right-8 if align right(3,6,9);

**y=0 if align top (7,8,9); y=size if align baseline(4,5,6); y=bottom if align bottom(1,2,3);

Der Wert -1 für x oder y ist die Mitte der entsprechenden Koordinatenachse.
