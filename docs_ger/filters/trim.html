Trim

Trim(clip, int "first_frame", int "last_frame")
Trim(clip, int "first_frame", int "-num_frames")

Mit Trim wird ein Clip geschnitten, so dass nur noch die Frames first-frame bis last-frame enthalten sind. Auch der Ton wird entsprechend geschnitten. Beachte, dass AviSynth mit 0 anfängt zu zählen.

Wenn du eine negative Zahl für num_frames angibst, werden die Frames first_frame bis first_frame + num_frames - 1 zurückgeben.

Wenn Null für last-frame eingesetzt wird, wird das als "bis zum Ende des Clips" interpretiert. Das bedeutet, nur den allerersten Frame zu erhalten, ist nur mit num-frames möglich.

Beispiele:

# Die ersten 100 Frames löschen:
Trim(100,-100)

# Das ist das gleiche wie Trim(100,199):
Trim(1000,-100)

# Ist NICHT dasgleiche wie {{Trim(0,0)}} (das würde den ganzen Clip zurückgeben):
Trim(0,-1)
