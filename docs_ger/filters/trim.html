Trim

Trim(clip, int first_frame, int last_frame)

Mit Trim wird ein Clip geschnitten, so dass nur noch die Frames first-frame bis last-frame enthalten sind. Auch das Audio wird entsprechend geschnitten Wenn Null für last-frame eingesetzt wird, wird das als "bis zum Ende des Clips" interpretiert.
Eine negative Argument von last_frame gibt die Frames von first_frame bis first_frame + (- last_frame-1) zurück. Dies ist auch die einzige Möglichkeit, den allerersten Frame zu erhalten.

  

Beispiel:
Trim(100,0)     # die ersten 100 Frames löschen
Trim(100,-100)  # das ist das gleiche wie Trim(100,199)
Trim(0,-1)      # den ersten Frame zurückgeben
