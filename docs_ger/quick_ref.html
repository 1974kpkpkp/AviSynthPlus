Abkürzungen:

Benannte Funktionsargumente werden in Anführungszeichen "arg" geschrieben und sind immer optional. Nichtoptinonale Argument sind in eckige [] Klammern gesetzt.
Mögliche Typen von nicht-clip-Argumenten sind int (Ganzzahl), float (Dezimalzahl), string (Zeichenkette) oder bool (Wahr/Falsch).
[yuy2] usw. gibt an, dass dieser Farbraumdenotes unterstützt wird. Guck dir auch die Farbraumkonvertierungs Filter an für mehr Informationen.
[v 2.04] gibt an, ab welcher AviSynth Version der Filter enthalten ist.
