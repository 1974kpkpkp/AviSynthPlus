Abkürzungen:

Benannte Funktionsargumente werden in Anführungszeichen "arg" geschrieben und sind immer optional.
Mögliche Typen von nicht-clip-Argumentn sind int (Ganzzahl), float (Gleitkommazahl), string (Zeichenkette) or bool (Wahr/Falsch).
[yuy2] usw. beschreibt die möglichen Farbformate für ein Filter. Siehe auch ConvertToRGB / ConvertToYUY2 / ConvertBackToYUY2 für Details.
[v 2.04] bezeichnet die Version von AviSynth, seit der das Filter verfügbar ist.

script Textdatei mit den Filtern und Befehlen
clip Videodatei (oder eigentlich die Variable, in der das Video gespeichert wird)
file Datei
frame ein Bild in einem Video
fps (framerate) Bilder pro Sekunde, für PAL immer 25 (Bildwiederholrate)
fade schwächer werden, also z.B. ausblenden
reduce vermindern
trim Abschneiden
argument die Eingangsgrössen für eine Funktion bzw. einen Filterbefehl
stack übereinander packen
number eine Zahl
string eine Zeichenkette (d.h. eine Aneinanderreihung von Buchstaben, Ziffern,...)
int eine Ganzzahl (Zahl ohne Komma)
float eine Gleitkommazahl (mit Komma)
splice zusammenfügen
aligned, unaligned ausgerichtet, nicht ausgerichtet
expression Ausdruck, Formel
return zurückkehren, Rückgabe eines Wertes
optional eine Variable, die angeführt sein kann aber nicht sein muss
default Standardwert
true wahr, Ja
false falsch, nein
error Fehler
plugin Erweiterungs-Funktion
field, interlaced

bei einem Interlaced-Bild sind zwei "field"-Bilder (die zeitlich 1/50 sec auseinander liegen) auf ein "frame"-Bild zusammengemischt, und zwar immer die geraden Zeilen vom einen und die ungeraden vom zweiten "field"-Bild.
Die zeitliche Auflösung ist also doppelt so hoch, allerdings mit der halben Zeilenanzahl, wobei die vertikale Position dieser Zeilen stimmt (die fields werden beim Abspielen im TV nicht nur doppelt so hoch gemacht sondern auch um eine halbe Zeile nach oben bzw. unten verschoben)
bob springen (nämlich gerade der Effekt, dass die fields eines Interlaced-Videos hoch- und runter springen, wenn man sie einfach hintereinander abspielt)
progressive das Gegenteil von Interlaced - alle 1/25 sec ein ganzes Bild
