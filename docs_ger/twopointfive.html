AviSynth 2.5


Neue Video Features:
  • Eine neuer Farbraum wurde eingeführt: eingebaute YV12 Unterstützung. Es werden YV12 Quellen durch DirectShowSource, AVISource und diverse MPEG Dekoder unterstützt.
    Für eine Beschreibung der Farbräume, die Avisynth verarbeiten kann, siehe hier.

Neue Audio Features:
  • Interne Unterstütztung für Fließkomma Samplingtiefen. AviSynth hat nun eine interne Verarbeitung von Fließkomma Samplingtiefen.
  • Eingebaute Unterstützung von 24 Bit Integer und 32 Bit Integer Samplingtiefen. Die Eingabe und Ausgabe von Samplingtiefen von höherer Genauigkeit wird unterstützt.
  • Mehrkanal Unterstützung. AviSynth unterstützt nun eine unbegrenzte Anzahl an Kanälen bei der Eingabe und Ausgabe.
  • Automatische Konvertierung der Samplingtiefe. Wenn ein Filter eine bestimmte Samplingtiefe nicht unterstützt, wird diese konvertiert in eine Samplingtiefe, die die Qualität möglichst erhält (meisten dann Dezimalzahl). Dezimalzahl Samplingtiefen werden dann automatisch in 16 Bit zurückkonvertiert, bevor sie ausgeben werden.

Neue interne Features:
  • Optmierungen. Viele grundlegende Features wurde optimiert und sind nun schneller als in den bisherigen Versionen.
  • Bessere Ausnutzung des Caches. Filter könnnen nun Filters können nun dem Cache-Managment Hinweise geben, ob deren Quelle zwischengespeichert werden soll oder nicht, so dass der Arbeitsspeicher effizienter genutzt werden kann.

Plugins:
  • Filter für AviSynth 1.0x und 2.0x funktionieren nicht mit 2.5. Sie werden in der Regel abstürzen, wenn man es dennoch versucht. Es gibt eine Überprüfung, ob ein passendes Plugin geladen wird.
  • Plugins müssen für AviSynth 2.5 neucompiliert werden !
  • Plugins für AviSynth 2.5 können natürlich nicht mit alten AviSynth Versionen genutzt werden.
  • Es gibt nun einen anderen Schlüssel in der Registrierung für das Verzeichnis, in dem man die Dll`s fürs automatische Laden von Plugins ablegen kann.
